# 📚 Library Management System

## Overview

This project is a SQL-based Library Management System developed as part of the SQL Developer Internship.

It manages:
- Authors
- Books
- Members
- Book Loans

---

## Features

- Store author information
- Store book information
- Manage library members
- Track borrowed books
- View book loan records
- Perform SQL JOIN queries
- Generate reports using SQL

---

## Database Tables

1. Authors
2. Books
3. Members
4. Loans
